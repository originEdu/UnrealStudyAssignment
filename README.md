# 함정 맵 기반 몬스터 AI 게임 과제
## 과제 진행시 문제 발생
1. 매쉬가 깨짐  
→ 원인은 파일이동으로 추정  
-> 복습겸 새로운 프로젝트로 아예 처음부터 시작
2. 캐릭터가 들고있는 총과 크로스헤어(컨트롤러) 중앙의 간극이 너무 심함   
-> ABP→ 애님그래프  
→ AO_Hip에 Yaw 끊어줘서 해결함  
→ 마우스 돌아가면 Yaw값 변경되는데 AO_Hip에서도 Yaw값을 변경해서 발생하는 문제로 파악함
3. 몬스터가 죽는 애니메이션없이 바로 사라짐  
→ 딜레이 넣어서 해결함
4. 비헤이비어트리에서 BTTask_Death로 구현했더니 Monster가 바로 사망하지 않음  
→ 여러가지 시도해봤지만 해결되지않아  
→ 다른 로직으로 변경  
→ Task를 사용하면서 바로 사망하게 만드는 로직은?
5. 몬스터가 공격할 때 끊김  
→ 애니메이션 시간만큼 wait 넣어서 해결함  
→ 좋은 방법은 아닌듯
6. 몬스터 레이케이스트 즉시터짐 = 회피불가  
→ attack 애니메이션에 notify 이벤트 → 함수 호출 → 해결  
→ 질문사항 발생  
→ 디버그가 안찍힌다
7.  타이머 매 프레임마다 로그찍힘 불편함  
-> Set Timer by Event 노드로 해결함
8.  몬스터 이동간격 5초  
-> wait걸어줘야하는거 맞는가요?
9.  anim 그래프에서 Idle Move  Run 상태   
→ 뛰다가 정지시 Run → Move → Idle인데  
→ Idle-Run연결없어도 자연스러움  
→ 엔진에서 해주는 건가요?
10. 어택, 데스  
→ ABP에서 상태관리로 변경하는 구조로 변경하다가 포기….
11. 몬스터 사망 시  
→ 1~5초 랜덤 시간 후 재스폰 ???  
→ tick에서 구현시 2마리 동시에 죽었을 경우, 늦게 죽은 몬스터 10후에 생성될 수 있음  
→ 함수화  
→ 함수에서 while 돌리면 안됨!!  
→ set timer by Event노드 사용 로직도 2마리 동시에 죽었을 경우 한마리 누락됨  
→ 몬스터 사망처리에서 리스폰로직 구현  
→ 몬스터 생성 안됨  
→ destroy때문인듯  
→ destroy전에 일단 몬스터 충돌+렌더링 제거   
→ delay후 몬스터리스폰함수 호출 후에 destory  
→ 해결
12. 플레이어 킬  
→ 약간 느리게 콘솔이 찍힘  
→ 몬스터 사망처리에 플레이어 킬 증가 로직이 있어서 그런듯
13. 몬스터 스폰 문제점 발생   
→ safe존에 생성이 됨  
→ 디버그해보니 RandomSpawnPoint가 내가 생각한 결과값이 안나옴  
→ 로직은 문제가 없음  
→ 랜덤으로 생성되는 값보다 실행핀이 먼저 실행되서 그런듯  
→ 지역변수 생성해서 해결함
14. 함정 벽 몬스터가 통과하는 문제 발생  
→ 프로젝트 세팅-콜리전에서 새로운 오브젝트 채널 Monster 생성  
→ 콜리전 프리셋에서 오브젝트 타입을 Monster로 변경함  
→ 생성한 벽의 오브젝트 반응에서 Monster만 block으로 설정  
→ 해결
15. 랜덤한 방향으로 이동하는 구체  for loop 생성시 seed가 고정되어 보임
